שלבפעולהפרטים

5. ארכיטקטורת המודל — 1D CNN

רכיבפירוט
Input(batch, 1, 128)
Conv1d ראשוני1 → 32 channels, kernel=1
Blocks4 × (Conv1d × 2 + skip + MaxPool1d), 32 channels, kernel=5
Flatten→ 160 features
Linear160 → 160 → 1
OutputSigmoid → הסתברות [0,1], threshold ≥ 0.5 = Abnormal
LossBinary Cross Entropy (BCELoss)

Batch Size32
Epochs30

12. מסקנות

  1. כל 5 דרישות הפרויקט ממומשות — סיווג בינארי, preprocessing, MIT-BIH, 3 splits, 1D CNN.
  2. המודל מתכלל — 84% accuracy על 5 מטופלים חדשים.
  3. Recall גבוה (93%) — מתאים כשחשוב לא לפספס חריגים.
  4. Precision בינוני (64%) — false positives; ניתן לשפר עם threshold tuning.
  5. Overfitting — המודל נבחר ב-epoch 5; epochs מאוחרים לא שיפרו generalization.
  6. המלצות: Early stopping, Dropout, threshold optimization, data augmentation.
